AgentForge

One spec, many agents

ATT&CKLens Benchmark

Can AI agents build defensible defensive tooling?

A reproducible, defensive cybersecurity benchmark: seven AI coding agents each built MITRE ATT&CK-aligned defensive tooling from the same spec, and every artifact was scored against a 100-point rubric — ATT&CK mapping discipline, prompt-injection resistance, evidence citation, uncertainty handling (“No Clear Mapping”), detection ideas, remediation guidance, and analyst follow-up questions.
