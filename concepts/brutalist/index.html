P/01

AgentForge

One spec, many agents. A configuration framework for agentic AI coding assistants — published to npm.

Write your project rules once; AgentForge compiles them into native configuration for six adapter targets: Claude Code, Codex, Gemini CLI, Cursor, Aider, and a generic fallback. No more six drifting copies of the same instructions.

It ships like software, not a gist: adapter round-trip tests, install smoke tests, a doctor CLI for diagnosing broken setups, a readiness runbook, and a live demo. MIT licensed.

P/02

ATT&CKLens Benchmark

Can AI coding agents build secure, uncertainty-aware, MITRE ATT&CK-aligned defensive tooling? I built a reproducible benchmark to find out.

Seven agent artifacts, one branch per agent, scored against a rubric of 100. Graded on ATT&CK mapping discipline, prompt-injection resistance, evidence citation, uncertainty handling (“No Clear Mapping”), detection ideas, remediation guidance, and the analyst follow-up questions each artifact asks.

Anyone can re-run the entire harness with benchmark-all.ps1 and check my grades. That is the point: an evaluation you cannot reproduce is an opinion.
