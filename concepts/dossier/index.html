EXHIBIT B

AgentForge ATT&CKLens Benchmark

Reproducible defensive-security benchmark for AI coding agents · MIT license

The question under examination: can AI coding agents build secure, uncertainty-aware, MITRE ATT&CK-aligned defensive tooling? Seven agent artifacts, one branch per agent, each scored against a 100-point rubric covering ATT&CK mapping discipline, prompt-injection resistance, evidence citation, uncertainty handling (“No Clear Mapping”), detection ideas, remediation guidance, and analyst follow-up questions. The harness re-runs end to end via benchmark-all.ps1.
