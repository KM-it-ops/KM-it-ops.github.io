# Pin repositories (30 seconds)

GitHub has **no API** to pin repos. This is the one profile step that must be done in the browser.

## Steps

1. Open https://github.com/KM-it-ops while logged in
2. In the right sidebar, click **Customize your pins**
3. Select exactly these six (in this order):

| # | Repository |
|---|------------|
| 1 | `AgentForge-ATT-CKLens-Benchmark` |
| 2 | `AgentForge` |
| 3 | `memory-mcp` |
| 4 | `security-log-anomaly-detection` |
| 5 | `phishing-email-classifier` |
| 6 | `KM-it-ops.github.io` |

4. Click **Save pins**

## Why this order

Leads with your differentiator (ATT&CKLens benchmark), then framework depth, MCP tooling, classic SOC proof, ML proof, and portfolio site.
