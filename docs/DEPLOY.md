# Deploying to GitHub Pages

This site is **static HTML, CSS, and JavaScript** with no build step.

1. Push `main` to `KM-it-ops/km-it-ops.github.io`.
2. In the repository **Settings → Pages**, set **Source** to **Deploy from a branch** and choose **`/ (root)`** on **`main`**.
3. Wait for the Pages build; the site is served at `https://km-it-ops.github.io/`.

The canonical **PDF résumé** is at `/assets/Mahmoud_Al_Kurdi_Resume_2026.pdf` (linked from the home page and profile README). The print-friendly **HTML résumé** is at `/resume/index.html` (use **Print / Save as PDF** in the browser if you need an alternate export).
