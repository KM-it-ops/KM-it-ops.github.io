# LinkedIn Setup Checklist

Complete these steps in the LinkedIn UI. Assets are in [`../assets/`](../assets/).

## 1. Headline

Copy from [`HEADLINE.md`](HEADLINE.md) — use the SOC-focused option.

## 2. About

Copy from [`ABOUT.md`](ABOUT.md).

## 3. Banner image

1. Open `assets/linkedin-banner.svg` in a browser (or export from Figma/Inkscape).
2. Export as **PNG, 1584 × 396 px**.
3. LinkedIn → **Profile → Edit banner → Upload**.

File: [`../assets/linkedin-banner.svg`](../assets/linkedin-banner.svg)

## 4. Profile photo

Use the same photo everywhere for brand consistency:

- Source: `assets/profile-portrait.png` (portfolio)
- Optional branded variant: `assets/masked-signal-photo.png`

LinkedIn → **Profile → Edit photo → Upload**.

Tips:

- Head and shoulders, dark/neutral background
- No filters; match the professional tone of the Masked Signal site
